1. data 폴더 : 전력수요와 기상 데이터의 raw 데이터를 포함해 전처리된 데이터(csv)를 담은 폴더
2. DLinear 폴더 : DLinear 모델의 학습 및 추론을 위한 소스코드 파일을 담은 폴더.
3. results : 최종모델의 output 값(npy)이 담긴 폴더.
4. weights_plot : 실험 과정의 모델과 최종 모델의 Weight Plot(jpg)이 담긴 폴더.
5. evaluate.ipynb : 실험 과정의 모델과 최종 모델에 대한 추론 및 평가, Weight Plot를 생성하는 소스코드 파일.
6. preprocessing : raw 데이터 전처리 소스코드 파일.
7. EDA : 데이터 EDA 소스코드 파일. 분석보고서 4페이지 탐색적 데이터 분석 내용.
